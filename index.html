MOTIVACION

Motivación

La generación automática de imágenes mediante Inteligencia Artificial se ha convertido en una de las aplicaciones más interessantes del Deep Learning. Los modelos generativos permiten crear imágenes nuevas a partir de patrones aprendidos durante el entrenamiento. En este proyecto se utiliza un modelo GAN (Generative Adversarial Network) entrenado con imágenes de CryptoPunks para generar personajes digitales únicos. El objetivo es comprender el funcionamiento de las redes generativas y analizar la capacidad de la IA para producir contenido visual original.

ESTADO DEL ARTE

Estado del Arte

Las Redes Generativas Antagónicas (GAN) fueron propuestas por Ian Goodfellow en 2014. Desde entonces, han sido ampliamente utilizadas para la generación de imágenes, videos y contenido sintético.

Entre los modelos más conocidos se encuentran:

  • DCGAN (Deep Convolutional GAN).
  • StyleGAN y StyleGAN2.
  • CycleGAN.
  • BigGAN.

Las GAN han sido utilizadas en la creación de rostros humanos, arte digital, videojuegos y colecciones NFT como los CryptoPunks. Actualmente representan una de las técnicas más importantes dentro del campo de la generación de imágenes mediante Inteligencia Artificial. El proyecto utiliza una arquitectura GAN preentrenada para generar imágenes de CryptoPunks.

Funcionamiento general

1
Se carga el modelo generador entrenado.
2
Se generan vectores aleatorios utilizando una distribución normal.
3
El generador transforma esos vectores en imágenes sintéticas.
4
Las imágenes generadas se almacenan en un archivo PNG.
5
Finalmente se visualizan utilizando Matplotlib.

CODIGO

Codigo del Modelo

Este es el codigo Python utilizado para generar los CryptoPunks con PyTorch y una GAN preentrenada.

Python 
import torch
from huggingface_hub import hf_hub_download
from torch import nn
from torchvision.utils import save_image
import matplotlib.pyplot as plt
import matplotlib.image as mpimg


class Generator(nn.Module):
    def __init__(self, nc=4, nz=100, ngf=64):
        super(Generator, self).__init__()
        self.network = nn.Sequential(
            nn.ConvTranspose2d(nz, ngf * 4, 3, 1, 0, bias=False),
            nn.BatchNorm2d(ngf * 4),
            nn.ReLU(True),
            nn.ConvTranspose2d(ngf * 4, ngf * 2, 3, 2, 1, bias=False),
            nn.BatchNorm2d(ngf * 2),
            nn.ReLU(True),
            nn.ConvTranspose2d(ngf * 2, ngf, 4, 2, 0, bias=False),
            nn.BatchNorm2d(ngf),
            nn.ReLU(True),
            nn.ConvTranspose2d(ngf, nc, 4, 2, 1, bias=False),
            nn.Tanh(),
        )

    def forward(self, input):
        output = self.network(input)
        return output


model = Generator()
weights_path = hf_hub_download('nateraw/cryptopunks-gan', 'generator.pth')
model.load_state_dict(torch.load(weights_path, map_location=torch.device('cpu')))
model.eval()

out = model(torch.randn(64, 100, 1, 1))
save_image(out, "punks.png", normalize=True)
print("Generated punks.png with 64 punks!")

img = mpimg.imread("punks.png")
plt.imshow(img)
plt.axis("off")
plt.title("Generated CryptoPunks")
plt.show()
CONCLUSIONES Y RECOMENDACIONES

Conclusiones y Recomendaciones

Conclusiones

  • Las GAN permiten generar imágenes nuevas y originales a partir de datos aprendidos durante el entrenamiento.
  • El modelo preentrenado de CryptoPunks demuestra que la Inteligencia Artificial puede reproducir características visuales complejas sin necesidad de diseñar manualmente cada imagen.
  • El uso de modelos preentrenados facilita la experimentación con Deep Learning sin requerir grandes recursos computacionales para el entrenamiento.

Recomendaciones

  • Utilizar una GPU para acelerar la generación y entrenamiento de modelos generativos.
  • Experimentar con diferentes vectores de ruido para obtener resultados más variados.
  • Analizar arquitecturas más modernas como StyleGAN para mejorar la calidad de las imágenes generadas.
  • Aplicar técnicas de generación de imágenes en otros dominios como arte digital, diseño gráfico o videojuegos.
